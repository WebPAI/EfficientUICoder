Canvas - Integration - Proctorio

Proctorio is approved for use within the Digital Learning Environment (DLE). This product requires an institutional license; contact your institutional Canvas administrator for more information.

Summary:

Proctorio allows Canvas quizzes to be proctored and reviewed either by a Proctorio representative or machine learning algorithms.  Proctorio consists of two main components.  First, settings enabled in Quizzes allow instructors and designers to create desired proctoring conditions.  Second, the Proctorio gradebook provides a listing of all students who have taken an exam, along with all recorded information. A product FAQ is available here: https://proctorio.com/frequently-asked-questions 

Costs:

This is not a free tool.  Institutions can purchase an institutional license or pay per exam.  Proctorio also supports a model in which students pay.  For more information, go to https://proctorio.com/ and select Get Started.

New Users:

Interested faculty and staff should contact their institutional admin if they are interested in using this tool.

Directions:

To start, install Proctorio's Google Chrome extension, as explained at https://proctorio.com/support. From there, work with your institutional administrator on how to access exam setup instructions from within Canvas.
